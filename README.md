# Digital Spectr Test Task

## Installation

- `$ git clone https://github.com/kaliyev-dev/-digital-spectr-test-task.git`
- `$ cd digital-spectr-test-task`
- `$ yarn` init packages

## Usage

- `$ yarn start` starts dev server
- `$ yarn build` builds prod version of app into `build/`
